# Role Permissions

## Super Admin

- Manage all centres
- Manage all users
- View all reports
- Configure application
- Billing
- Subscription

---

## Centre Admin

- Manage teachers
- Manage classrooms
- Manage children
- Manage admissions
- Manage fees
- View reports
- View attendance

---

## Teacher

- Mark attendance
- Complete daily workflow
- Capture memories
- Write teacher note
- View assigned classrooms only

---

## Parent

- View linked children
- View today's highlights
- View memories
- Read teacher notes
