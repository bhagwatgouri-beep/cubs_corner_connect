---

# Purpose

This document is the single source of truth for Cubs Corner Connect.

Every developer, AI assistant, designer, tester and future team member should understand the project by reading this document before making any changes.

Whenever there is a conflict between code and this document, this document represents the intended product direction until officially updated.

---

# Our Mission

Build the most teacher-friendly daycare application.

Every feature must satisfy at least one of the following:

• Save teacher time.

• Give parents peace of mind.

If a feature does neither, it should not be included in the Launch Edition.

---

# Our Vision

Teachers should spend their time with children.

The application should quietly document the day.

Parents should always feel informed without feeling overwhelmed.

Technology should support relationships, not replace them.

---

# Product Principles

Children First.

Teachers Teach.

The App Documents.

Default + Exceptions.

Simple.

Honest.

Useful.

Warm.

Trustworthy.

Professional.

---

# Launch Goal

Launch Cubs Corner Connect in real classrooms on **12 August 2026**.

Success is defined as:

A teacher can complete an entire school day using the application.

Parents can confidently understand how their child spent the day.

The application reduces teacher workload instead of increasing it.
