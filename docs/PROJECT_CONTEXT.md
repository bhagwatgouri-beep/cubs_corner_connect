# Cubs Corner Connect

Version: 1.0

---

# Project Overview

Cubs Corner Connect is a daycare management application developed by Bhagwat Education Trust.

The purpose of the application is to reduce teachers' documentation workload while keeping parents informed about their child's day in a simple, warm and trustworthy manner.

---

# Vision

Teachers should spend time with children.

The app should quietly document the day.

---

# Mission

Build the most teacher-friendly daycare application.

Every feature should either:

• Save teacher time

or

• Give parents peace of mind.

If it does neither, it should not be built.

---

# Target Users

1. Parents

Need reassurance.

Need meaningful updates.

Need photos.

Need trust.

---

2. Teachers

Need speed.

Need simplicity.

Need large buttons.

Need minimal typing.

---

3. Administrator

Needs complete visibility.

Needs reports.

Needs attendance.

Needs classroom management.

---

# Core Philosophy

Children First.

Teachers Teach.

The App Documents.

Default + Exceptions.

No unnecessary data entry.

No duplicate workflows.

---

# Product Principles

Parents should never wonder if their child is okay.

Teachers should never repeat the same work twice.

Every teacher action should create value for parents.

The application should feel calm, trustworthy and warm.
